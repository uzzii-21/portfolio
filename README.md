Built with:

- [Tailwind CSS](https://tailwindcss.com/)
- [TypeScript](https://www.typescriptlang.org/)

## Getting Started

1. Install dependencies

   ```bash
   yarn
   ```

2. Fire up the server and watch files

   ```bash
   yarn dev
   ```

## Build

1. Build

   ```bash
   yarn build
   ```

2. Build preview!

   ```bash
   yarn start
   ```
